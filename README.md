# Iris Flower Classification

## Overview
This project uses Machine Learning to classify Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

The model predicts the flower species based on sepal and petal measurements.

## Technologies Used
- Python
- Scikit-learn
- NumPy
- Pandas

## Dataset
The Iris dataset is loaded directly from Scikit-learn and contains 150 flower samples with 4 features.

## Model
A Random Forest Classifier is used to train and predict the flower species.

## How to Run
```bash
pip install -r requirements.txt
python app.py
```

## Output
- Flower species prediction
- Accuracy score
- Classification report
- Confusion matrix
